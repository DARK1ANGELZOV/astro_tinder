# AstroVinci — Telegram-бот знакомств в астротематике

Астро Тиндер — асинхронный Telegram-бот на Aiogram 3, который помогает пользователям знакомиться внутри одного города. Бот собирает анкету, показывает кандидатов с астрологией, поддерживает лайки, взаимные матчи и переписку через бота или напрямую по username.

**Топ-фичи**
- Пошаговая регистрация анкеты с проверкой возраста 18+ и настройкой предпочтений.
- Медиа-профиль: до 3 фото или 1 видео, предпросмотр анкеты перед сохранением.
- Поиск кандидатов по городу, полу и диапазону возраста; исключение уже просмотренных и лайкнутых профилей.
- Лайки и взаимные матчи с автоматическими уведомлениями обеим сторонам.
- Обмен сообщениями через бота, а при наличии `username` — быстрые ссылки на прямой чат.
- Просмотр и редактирование своей анкеты, включая быстрое обновление описания или полную перерегистрацию.
- Команда `/language` для выбора языка (RU/EN/UK/ID) и хранение `lang_code`.
- Обратная связь модераторам через `/feedback`.
- Логи с ротацией и архивированием в `logs/`.

**Как это работает**
1. Пользователь запускает `/start` и получает приветствие и ссылку на отдельного бота «Астро Совместимость».
2. Проходит регистрацию: возраст, пол, предпочтения, возрастной диапазон, имя, город, Instagram, описание и медиа.
3. Смотрит предпросмотр и подтверждает анкету.
4. Просматривает анкеты, ставит лайки или пропускает, может запросить все медиа профиля.
5. При взаимной симпатии бот создаёт матч и даёт варианты общения.

**Команды**
- `/start` — запуск и вход в регистрацию или показ существующей анкеты.
- `/language` — выбор языка интерфейса.
- `/feedback` — отправка сообщения модераторам.

**Запуск локально**
1. Установите зависимости через Poetry.
```powershell
poetry install
```
2. Создайте или обновите `.env` с переменными окружения.
```env
BOT_TOKEN=your_bot_token
MODERATOR_ID=your_telegram_id
DATABASE_URL=sqlite+aiosqlite:///./database.sqlite3
```
3. Запустите бота.
```powershell
poetry run python main.py
```

Таблицы в БД создаются автоматически при старте. Чтобы сбросить БД, удалите `database.sqlite3`.

**Конфигурация**
- `BOT_TOKEN` — токен Telegram-бота из BotFather.
- `MODERATOR_ID` — Telegram ID модератора для `/feedback`.
- `DATABASE_URL` — опциональный URL SQLAlchemy (по умолчанию SQLite в корне проекта).

**Стек**
- Python 3.11+
- Aiogram 3
- SQLAlchemy Async + aiosqlite
- Loguru
- dotenv

**Структура проекта**
- `main.py` — точка входа, запуск бота и инициализация БД.
- `app/handlers/` — бизнес-логика и обработчики команд.
- `app/database/models.py` — ORM-модели `users`, `likes`, `matches`.
- `app/database/requests.py` — операции чтения и записи в БД.
- `app/keyboards/` — клавиатуры и кнопки.
- `app/utils/` — FSM-состояния и логирование.
- `logs/` — ротационные логи.
- `database.sqlite3` — файл БД SQLite по умолчанию.

**Примечания**
- Бот не запрашивает паспортные данные; в интерфейсе есть дисклеймер о безопасном общении.
- В карточках присутствует текст о «совместимости»; вычисление совместимости вынесено в отдельный бот по ссылке в `/start`.
